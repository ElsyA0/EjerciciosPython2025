# EjerciciosPython2025
En este entregable se desarrollar un conjunto de ejercicios básicos en Python con el objetivo de reforzar conceptos clave de programación.
Ejercicios a Desarrollar
1. Número par o impar
Crea un programa que:
•	Pida al usuario un número entero.
•	Determine e imprima si el número es par o impar.
2. Números naturales con ciclo while
Desarrolla un programa que:
•	Solicite al usuario un número entero positivo.
•	Imprima todos los números naturales desde 1 hasta el número ingresado utilizando un ciclo while.
3. Cálculo de descuento
Define una función que:
•	Reciba como parámetros un precio original y un porcentaje de descuento.
•	Devuelva el precio final después de aplicar el descuento.
•	Puedes agregar validaciones básicas si lo deseas (opcional pero recomendable).
4. Conteo de pares e impares
Crea un programa que:
•	Genere una lista de números aleatorios (tú decides la cantidad, por ejemplo, 20).
•	Cuente cuántos de esos números son pares y cuántos son impares.
•	Muestra los resultados al usuario.
5. Secuencia de Fibonacci con recursión
Implementa una función recursiva que:
•	Reciba un número n.
•	Devuelva los primeros n términos de la secuencia de Fibonacci.
•	Puedes imprimirlos o devolverlos en una lista.

Forma de entrega 
•	Sube un solo archivo llamado:
•	Apellido_Nombre_EjerciciosPython.py
•	(un solo archivo facilita la revisión).
                 # ---- EJERCICIO 1: PAR O IMPAR ----
                 # ---- EJERCICIO 2: WHILE ----
•	...y así sucesivamente.
•	No entregues carpetas, ni ZIP, ni múltiples archivos. Solo un .py.
•	Si prefieres organizarlo por funciones o secciones, está perfecto mientras sea un mismo archivo final.
